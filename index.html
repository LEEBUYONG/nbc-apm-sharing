🔔 퀴즈 골든벨

운영·디자인·AI 문제를 풀어보세요!

⚠️ 입력 규칙 (중요!)

① 숫자만 입력 문제 → 단위(%, 시, 일 등) 빼고 숫자만!

② 한글 입력 문제 → 힌트에 표시된 글자 수 그대로 입력

③ 영어 입력 문제 → 대소문자 구분 없음 (CMYK / cmyk 모두 정답)

④ 띄어쓰기 → 띄어쓰기 여부 상관없이 인정

⚠️ 한 글자라도 다르면 오답 처리!

🎮 진행 방식

1
ZEP 미니게임으로 문제 출제
2
정답 입력 (15~20초)
3
정답 공개 & 오답자 자동 탈락
4
최종 생존자 1~3명 확정
